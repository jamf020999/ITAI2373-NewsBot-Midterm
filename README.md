## Course

## Project Description
This project implements a complete NewsBot Intelligence System using Natural Language Processing techniques.

The system performs:
- Text preprocessing
- TF-IDF feature extraction
- POS tagging
- Dependency parsing
- Sentiment analysis
- Text classification
- Named Entity Recognition
- Comprehensive news analysis
